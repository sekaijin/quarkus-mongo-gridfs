== Installation

If you want to use this extension, you need to add the `io.quarkiverse.mongo-gridfs:quarkus-mongo-gridfs` extension first to your build file.

For instance, with Maven, add the following dependency to your POM file:

[source,xml,subs=attributes+]
----
<dependency>
    <groupId>io.quarkiverse.mongo-gridfs</groupId>
    <artifactId>quarkus-mongo-gridfs</artifactId>
    <version>{project-version}</version>
</dependency>
----
